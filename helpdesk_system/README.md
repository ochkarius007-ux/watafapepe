Helpdesk система МООНД - обработка технических заявок через веб-интерфейс, Telegram и MAX ботов.

## ШАГ 1: Установите необходимые пакеты

Откройте терминал и выполните команду:

```bash
sudo apt update && sudo apt install -y python3 python3-pip python3-venv git
```

## ШАГ 2: Перейдите в папку проекта

```bash
cd /workspace/helpdesk_system
```

## ШАГ 3: Создайте виртуальное окружение

```bash
python3 -m venv venv
```

## ШАГ 4: Активируйте виртуальное окружение

```bash
source venv/bin/activate
```

После активации вы увидите `(venv)` в начале строки терминала.

## ШАГ 5: Установите зависимости

```bash
pip install -r requirements.txt
```

## ШАГ 6: НАСТРОЙТЕ ТОКЕНЫ БОТОВ (ОБЯЗАТЕЛЬНО!)

Откройте файл config.ini для редактирования:

```bash
nano config.ini
```

Замените заглушки на реальные токены:

### Для Telegram бота:
1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/newbot`
3. Придумайте имя и username для бота
4. BotFather выдаст вам токен (например: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz...`)
5. Скопируйте этот токен
6. В файле config.ini найдите строку `bot_token = YOUR_TELEGRAM_BOT_TOKEN_HERE` в секции [TELEGRAM]
7. Замените `YOUR_TELEGRAM_BOT_TOKEN_HERE` на ваш токен

### Для MAX бота:
1. Войдите в административную панель MAX
2. Перейдите в раздел "Bots" или "API"
3. Создайте нового бота
4. Скопируйте полученный токен
5. В файле config.ini найдите строку `bot_token = YOUR_MAX_BOT_TOKEN_HERE` в секции [MAX]
6. Замените `YOUR_MAX_BOT_TOKEN_HERE` на ваш токен

### Сохраните файл:
- В nano: нажмите `Ctrl+O`, затем `Enter`, затем `Ctrl+X`

## ШАГ 7: Запустите основное приложение

В активированном виртуальном окружении выполните:

```bash
python3 app.py
```

## ШАГ 8: Откройте веб-интерфейс

Откройте браузер и перейдите по адресу:
```
http://localhost:5000
```

## ШАГ 9: Запустите ботов (в ОТДЕЛЬНЫХ окнах терминала)

Откройте НОВОЕ окно терминала для каждого бота.

### Для запуска Telegram бота:
```bash
cd /workspace/helpdesk_system
source venv/bin/activate
python3 bots/telegram_bot.py
```

### Для запуска MAX бота:
```bash
cd /workspace/helpdesk_system
source venv/bin/activate
python3 bots/max_bot.py
```

## ШАГ 10: Проверка работы

1. **Веб-интерфейс:** Откройте http://localhost:5000
   - Логин администратора: `admin`
   - Пароль: `admin123`

2. **Telegram бот:** Найдите вашего бота в Telegram и нажмите `/start`

3. **MAX бот:** Настройте webhook в панели MAX на адрес `http://ваш-сервер:5001/max/webhook`

## Частые проблемы и решения

| Проблема | Решение |
|----------|---------|
| Ошибка при установке пакетов | Убедитесь, что установлены python3-venv и python3-pip |
| Бот не отвечает | Проверьте правильность токена в config.ini и убедитесь, что Flask приложение запущено |
| Порт 5000 уже занят | Измените порт в config.ini в секции [FLASK] |
| ModuleNotFoundError | Убедитесь, что активировано виртуальное окружение и установлены все зависимости |

## Функционал

### Веб-интерфейс
- Регистрация и авторизация пользователей
- Создание и просмотр заявок
- Переписка в рамках заявки до закрытия
- Назначение исполнителей
- Изменение статуса заявок
- Фильтрация по статусам

### Telegram и MAX боты
- Автоматические ответы с предупреждением о персональных данных
- Создание заявок через чат
- Добавление сообщений к заявкам
- Проверка статуса заявок

## Важное уведомление

Все автоматические ответы ботов содержат предупреждение:

> ⚠️ **Важно:** Заявки с персональными данными не будут обрабатываться. 
> Пожалуйста, укажите в общих чертах, что требуется сделать. 
> Специалист сразу свяжется с вами по указанному телефону.
